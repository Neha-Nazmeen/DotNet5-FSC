CSS3 Exercises — Local Community Event Portal

This folder contains an HTML demo and external CSS implementing the requested exercises. Files:

- index.html — Demo page that shows usage of inline, embedded and external styles; includes examples for selectors, typography, forms, tables, multi-column text, and responsive media queries.
- styles.css — External stylesheet with labeled sections and comments per exercise requirements.

How to use:
- Open `index.html` in a browser.
- Use DevTools (F12) to inspect styles, simulate devices, and check that `styles.css` is loaded in the Network tab.

Notes on coverage:
- Exercise 1: Inline style applied to the main H1; embedded `<style>` in the head defines body background; `styles.css` holds reusable styles with comments.
- Exercise 2: `styles.css` includes comments and consistent indentation.
- Exercise 3: Examples of universal, element, ID, class and grouping selectors are present.
- Exercise 4: Uses HEX and RGBA colors, body has a fallback background color and a background image; section headers use linear-gradient.
- Exercise 5: Google Font included via `<link>` and font properties applied.
- Exercise 6: Link pseudo-classes and list styling implemented.
- Exercise 7: Table styling with zebra striping implemented.
- Exercise 8: Box model spacing, outline for inputs, visibility helper classes included.
- Exercise 9: Multi-column layout applied to `.news-content`.
- Exercise 10: Media query at 768px stacks nav and adjusts layout; grid/flex used for responsive layout.
- Exercise 11: Use DevTools to debug as usual; `styles.css` is referenced in the HTML and should appear in Network tab.
