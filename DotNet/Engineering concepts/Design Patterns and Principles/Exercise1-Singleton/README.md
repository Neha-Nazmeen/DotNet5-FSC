# Exercise 1 — Singleton Pattern

Project: SingletonPatternExample

This small example demonstrates a thread-safe Singleton Logger implementation in Java and a short test that verifies only one instance is used.

How to compile & run (from the project root):

# Compile
javac -d out src\main\java\com\example\singleton\*.java

# Run
java -cp out com.example.singleton.TestSingleton
